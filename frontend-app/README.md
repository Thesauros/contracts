# frontend-app

Static frontend concept for visualizing fund movement between chains, vault contracts and provider contracts.

## Run

Preferred:

```bash
cd frontend-app
npm run start
```

Then open `http://localhost:4173`.

Alternative:

- Open [index.html](/Users/ivanborisov/Desktop/thesauros/contracts/frontend-app/index.html) directly in a browser.

## What is included

- Dark visual language with motion inspired by `stargate.finance`
- Hard-edged modular cards inspired by `symbiotic.fi`
- Interactive scenario switching
- Cross-chain routing layer
- Arbitrum deployment addresses from `deployments/arbitrumOne/custom/current-deployment.json`
